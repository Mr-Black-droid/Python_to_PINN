# Python_to_PINN

## 6-Month Weekly Progress Structure

This repository now includes a `weekly_progress/` directory with weekly folders for the next 6 months:

- `Month1_Week_1` through `Month1_Week_4`
- `Month2_Week_1` through `Month2_Week_4`
- `Month3_Week_1` through `Month3_Week_4`
- `Month4_Week_1` through `Month4_Week_4`
- `Month5_Week_1` through `Month5_Week_4`
- `Month6_Week_1` through `Month6_Week_4`

Each weekly folder contains a `daily_progress.md` file to record day-by-day tasks and progress for that week.

When one week ends (for example `Month1_Week_1`), continue tracking in the next week's file (`Month1_Week_2/daily_progress.md`).
